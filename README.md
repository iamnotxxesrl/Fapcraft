# Fapcraft 🌸

**Fapcraft** is the a unofficial website for the **Fapcraft Discord server** — a fun, chaotic, and occasionally unhinged community space. It also includes details about our custom **Minecraft server**, featuring adult-themed mods and unique gameplay.

> 🔞 NSFW Community – You must be 18+ to join.

---

## 🌐 What’s Inside?

- 💬 Info and invite to our [Discord server](https://discord.gg/ahu7FUJMpR)
- 🧱 Details on our custom modded Minecraft server
- 📜 Rules, guidelines, and community updates
- 🌈 Smooth UI built with **Vite + TailwindCSS + TypeScript**

---

## 🛠 Tech Stack

- **Frontend**: Vite, React, TailwindCSS, TypeScript
- **Backend**: Drizzle ORM, Node.js (with serverless support)
- **Deployment**: Vercel
- **Tooling**: pnpm, Prettier, ESLint
